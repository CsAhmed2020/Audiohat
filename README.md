# Audiohat
Android Audio Player App

## Features:
- Audio player and controller
- Search for Audios
- Customize image and audio view

## Tools:
- Jetpack compose
- clean architecture
- MVVM
- dagger Hilt
- Media Service
- ExoPlayer
- MediaPlayerNotificationManager
